# 🧮 Numpy-Lab

Numpy-Lab is a **numerical computation web application** built using **Python, NumPy, and FastAPI**.  
It provides mathematical tools for performing symbolic and numerical calculations through a simple web interface.

The project is designed as a **learning and experimentation lab for numerical methods**.

---

## 🚀 Features

- First Derivative Calculation
- Second Derivative Calculation
- Numerical Integration
- Mathematical Expression Parsing
- Graph Visualization
- Fast API-based backend
- Interactive frontend interface

---

## ⚙️ Installation

Clone the repository:


git clone https://github.com/your-username/Numpy-Lab.git


Move into the project folder:


cd Numpy-Lab


Create a virtual environment:


python -m venv .venv


Activate the environment:

**Windows**


.venv\Scripts\activate


Install dependencies:


pip install -r requirements.txt


---

## ▶️ Run the Application

Start the FastAPI server:


uvicorn main:app --reload


Open in browser:


http://127.0.0.1:8000


API Documentation is available at:


http://127.0.0.1:8000/docs


---

## 📊 Example Usage

Enter a mathematical expression such as:


x3 + 2*x2 + x


The system will compute:

- First derivative
- Second derivative
- Numerical calculations
